# Khan Barbershop - Упрощённая админ панель

Мини-admin панель для управления Khan Barbershop - чатботом парикмахерской.

## 🎯 Функциональность

Админ панель содержит **4 основных раздела**:

### 1. 📊 Панель управления (Dashboard)
- **Статистика за день**: записей, диалогов, всего клиентов, рейтинг бота
- **Недавние записи**: список последних 5 записей клиентов
- **Активность чатов**: статистика обработанных сообщений, среднее время ответа

### 2. 💬 Логи переписки (Logs - Tab 1)
- **Групповой список чатов** слева (новые чаты сверху)
- **Подробный просмотр** всех сообщений в конкретном чате (новые сообщения сверху)
- **Поиск** по номеру телефона
- **Скачивание** логов в JSON формате
- **Модальное окно** с деталями каждого сообщения

### 3. 🤖 Логи бэкенда (Logs - Tab 2)
- **Список всех логов** системы с временными метками
- **Фильтрация** по уровню (ERROR, WARNING, INFO, DEBUG)
- **Цветовая индикация** уровня логирования
- **Сортировка** (новые логи сверху)
- **Скачивание** всех логов в текстовый формат

### 4. ⚙️ Настройки рассылки (Broadcast Settings)
- **Конфигурация** параметров автоматической рассылки (заглушка)
- **Управление номерами** для рассылки
- **Шаблоны сообщений** с поддержкой переменных
- **Расписание** отправки
- **Статус системы** рассылки
- **История** отправленных рассылок

## 🚀 Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка зависимостей
```bash
cd frontend
npm install
```

### Запуск dev сервера
```bash
npm run dev
```
Приложение запустится на `http://localhost:5173`

### Production build
```bash
npm run build
```
Готовая сборка будет в папке `dist/`

## 🔑 Авторизация

Панель защищена JWT авторизацией. По умолчанию используется учетная запись:
- **Username**: admin
- **Password**: admin123

Токен сохраняется в localStorage и используется для всех API запросов.

## 🌍 API подключение

Фронтенд подключается к бэкенду через переменную окружения:

**`.env` файл:**
```env
VITE_API_URL=http://localhost:8000
```

API базовый URL: `http://localhost:8000/api/admin`

## 🐛 Troubleshooting

**Проблема**: "Failed to resolve import"  
**Решение**: Запустите `npm install` для установки зависимостей

**Проблема**: API недоступен  
**Решение**: Проверьте значение `VITE_API_URL` в `.env` файле и что бэкенд запущен

**Проблема**: Логи не загружаются  
**Решение**: Убедитесь, что на бэкенде есть endpoint `/api/admin/backend-logs`
